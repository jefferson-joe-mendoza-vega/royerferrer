# web_royer

Sitio modularizado para la campana de Royer Ferrer.

## Estructura

- `index.html`: estructura visual principal
- `assets/css/styles.css`: estilos globales propios
- `assets/js/main.js`: punto de entrada de JavaScript
- `assets/js/modules/menu.js`: menu lateral movil
- `assets/js/modules/share.js`: boton compartir
- `assets/js/modules/joinForm.js`: formulario hacia WhatsApp
- `assets/js/modules/modal.js`: modal inicial con temporizador/video
- `assets/js/modules/aos.js`: inicializacion de animaciones AOS

## Uso

1. Abre `web_royer/index.html` en navegador.
2. Asegurate de mantener en la misma carpeta:
   - `logo.png`
   - `image.png`
   - `image2.png`
   - `video.mp4`
